Garmin Run Dynamics

Import a Garmin Connect activities export (Activities.csv) to backfill your run history. New activities are saved to your account and de-duplicated by start time, so you only ever import what's new. Stats below are computed from your full stored history — ground contact time, vertical oscillation, cadence, stride length and derived efficiency metrics — over default windows and any custom date range.

How to export your activities from Garmin Connect
  1. On a computer, open Garmin Connect → Activities → All Activities and sign in.
  2. Use the filter controls at the top to set Activity Type → Running (optional) and the date range you want — e.g. a full season, or just since your last upload.
  3. Scroll to the bottom of the list so every activity in that range loads on screen. The export only includes what is currently loaded.
  4. Click Export CSV (top-right of the activities list) to download Activities.csv.
  5. Drop that file below. Runs are de-duplicated by start time, so re-uploads only add what is new.
Garmin's CSV export is capped at ~1,000 activities and exports only the rows loaded on screen. For long histories, export in chunks by date range — each chunk merges in automatically.
